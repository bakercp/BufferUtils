Detailed Description

A class for printing into a raw data byte buffer.

Implements the Arduino Print interface. Is similar to a BufferWriter.

See also
BufferWriter
https://github.com/arduino/Arduino/blob/master/hardware/arduino/avr/cores/arduino/Print.h
https://www.arduino.cc/en/Serial/Print
