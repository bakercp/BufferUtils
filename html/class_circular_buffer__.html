Detailed Description

template<typename Type>
class CircularBuffer_< Type >

A templated class representing circular access to an array.

Circular buffers are also known as ring buffers or cyclic buffers.

Template Parameters
TypeThe data type of the wrapped array.

◆ Status

template<typename Type >
enum CircularBuffer_::Status

The possible return status after get, put, and peek operations.

Enumerator
STATUS_OK 

The operation was successful.

STATUS_OVERFLOW 

The head position overwrite the tail position.

This might indicate possible data loss.

STATUS_UNDERFLOW 

A value was read, but there was no unread value available.

This might indicate that the data read is invalid.

STATUS_INVALID 

The operation was invalid, e.g. an empty data buffer.

◆ get()

template<typename Type >
Status CircularBuffer_< Type >::get ( Type &  data)
inline

Get the data from the tail and advance the tail position.

If the underlying data is invalid or of size zero, STATUS_INVALID will be returned.

If the tail does not overtake the head, STATUS_OK will be returned. If the tail overtakes the head, then STATUS_UNDERFLOW will be returned, and neither the head or the tail will be incremented.

template<typename Type >
Status CircularBuffer_< Type >::put ( const Type &  data)

Insert data at the head position and advance the head position.

If the underlying data is invalid or of size zero, STATUS_INVALID will be returned.

If the head does not overtake the tail, STATUS_OK will be returned. If the head overtakes the tail, then STATUS_OVERFLOW will be returned, the tail will be incremented and overwritten by the head.
